这个十年，我撇下了青春！

发表于 2015-12-27 | 更新于 2019-01-20

再过几天就满30岁了，回想起自己真正懂事独立以来这十年（20-30），我不能留恋，似乎也没有很多可供怀念。

回首的时候总是觉得时间过得很快。

  • 想想我20岁的时候对自己的未来充满信心，30岁的时候却是一片迷茫；

  • 20岁有女朋友，30岁没女朋友；

  • 20岁觉得自己年轻，30岁了还是觉得年轻；

  • 20岁没有梦想，30岁依然没有；

自参加工作08年后我每年都会给写一个总结，总结自己的生活工作和爱情。

从09年的充满激情，到10年的坚定，再到11年的失控，12年的迷茫，13年的累，以及到14年无可奉告，每一年我都会审视一次自己，在反思中去寻找成长的突破点，是的，经过8年的磨练，我总算是长大了，至少清楚地认识了自己是一个什么样的人，适合做什么样的事。

思想上，我并不是个伟人，没有什么宏伟的目标，我很普通，只是希望自己能过得好、家人能过得好、下一代能过得好，为了这样的目标，我不得不努力，去了解这个世界、这个社会，以便寻找自己的立足之地。作为一个年轻人，我对现在的社会政治是十分不满的，可能很多人这时候要说我是愤青了，管什么社会形态，你好好做好你的公务员，努力挣钱养家去吧！他们会如此说。但真正的年轻人，不关心社会那就是不关心自己的下一代。我不知道看到我这篇文章的有多少人会明白，目前的国外是如何看待我们中国，是如新闻联播里面说的那样：人民安居乐业幸福生活喜迎油价上涨，看看最近北京的雾霾，看看每一次发生重大事情后的处理结果（其实很多事情最后根本没有结果：就如今年的长江沉船、天津爆炸，这么久过去了，依然是一团迷雾），这个社会病了，还病得不轻！而我，还正是处在这个运转机构里面最基层的一颗螺丝。世界上的专制国家剩下不到几个了，不能访问google、facebook 的国家除了朝鲜就还只剩下我们，当大家都在嘲笑朝鲜的时候，为什么不反思一下我们自己的祖国。我不想在这里多说了，我这么担忧那是因为我理智地爱着这个国家。其实，关心国事，那是因为我想要我的下一代能生活得更好，所以在根本上我是自私的。

谈过不少恋爱，却依然不能好好爱一个人，也许是我木讷，不知道如何讨得女孩子欢心；也许是我太过算计，总想把一个人看得清楚。朋友开玩笑，说每一个和我恋爱过的女孩子马上就能结婚，我都成女孩结婚前的进修学校了。确实，每个喝和我分手的女孩都会在一年内结婚，所以，如果有哪个女孩想快点结婚的话，那么——>来和我谈一场恋爱吧！曾经和一个女孩说“我没法给你我全部的爱，最多给你80%，还有20%我要爱自己”，现在想想真傻逼，撒个谎哄女孩子开心一下也好啊！可我就是这么实诚和理性，后来我明白了，感情中的甜言蜜语是可以修辞的，那不叫谎话！曾写过一篇《屌丝男求偶指南》，12年的时候吧，那时候还天真地以为自己30岁肯定能结婚，这么一眨眼就30了，还是孤家寡人一个。

说实在，我现在一点也不期待爱情，反而是觉得单身让我变得越来越好，我明白那是一种责任，也想过如何去承担起那份责任，大学的同学不知道还记不记得我们毕业聚餐的那个晚上我在广场下的国旗下所说的责任，是的，我是个愿意承担责任的人，但是我发现自己还不够成熟，我依然还有玩心，我还没为组建家庭积累好必要的财富，我是个贫一代，我父母都是很农民，所以我很努力地工作，除了做公务员的工作，我还努力地学习编程，开自己的公司，为的是挣钱，能够让父母、让自己和将来的老婆孩子过得更好。

下一个十年，我会去哪里？

都说30-40岁这十年是人生最重要的十年，那么，下一个十年，我要去哪儿，我会变成什么样？为了认真对待这个问题，我特地去知乎寻找答案。看了两个问题，一个是《如果一个人在监狱呆了20年，出来后会和社会脱节吗？》，另一个是《30岁可以做什么，可以在5年后受益匪浅》，看第一个问题是因为我觉得现代社会变化如此之快，我要如何才不被社会的洪流所淘汰，得到的结论是，只要你有一项过人的技能，以及很好的人脉关系，你就能很好跟上社会的步伐。而第二个问题，其实是我自己想问的，看了一个知友超过6000赞的回答，我认为，不论做什么，只要你有足够的勇气和智力，不懒惰，你都能在某一方面成为专家，他举了个例子，一个人在40岁的时候想写小说，本来是学理科，然后用5年时间读书，5年时间练笔，终于在第10年的时候开始写小说，并获得大卖。

所以，这个时候我应该沉淀自己，学习是亘古不变的驱动力，也许30岁改行还不算晚吧！很久没能静下心来写下这么多文字了，微博和朋友圈的只言片语只能表达我当时的一点点心情，而沉下心来回忆更长久的时光不仅仅是能够取暖，更重要的是总结、沉淀，为了过好将来那值得期待的时光。

领导和下级分别是什么东西？

发表于 2015-02-04 | 更新于 2019-01-20

本文来自知乎。首先说，这是一个非常好的问题。请各位对职场话题有兴趣的人一定要看这个答案。因为这个答案是我们谈一切职场题目的基础之一。1，领导（上级）是个什么东西？一个基本概念是，公司业务无民主。总经理由董事会授权管理公司，在法律允许的范围内，在不违反职业道德的前提下，总经理对公司的业务和管理有绝对的权利。管理层由总经理授权分管相关业务端口，有绝对的权利。部门经理由管理层授权管理相关业务部门，有绝对的管理权力。这区别于社会的组织方式，国家的首领是民选的，是全体公民赋予其权利以管理国家，但公司不是。而你的总经理，以及你的领导，不是民选的，是任命的，是由层层授权来获得权利管理公司的。因此，在最底层的理论上讲，领导对你的管理行为，包括工作安排和对工作结果的评价，在不涉及人格侮辱、不违反职业道德、不违反法律、不超越授权的前提下，是绝对的，甚至可以是主观的。2，下级是个什么东西？八个字：拿人钱财，与人消灾。在多数的情况下，你的领导也是某人的下级。大家都应秉持这个原则。即使我们叫打工仔也好，职业经理人也好，工作这件事，就是因为你拿了或者要拿别人的钱，然后你要把别人安排的事情做好。今天你来上班了，没有人安排给你工作，但是你的时间被占了，对方也应该付给你钱（工资）。明天给你安排工作，你有可能做好，也有可能没做好（好与不好可能是个上级主观评价），那么这会决定你奖金的多少。所以，拿人钱财，与人消灾（还有下半句，以后别的题再说）。这是最基本的职业精神和职业道德。3，什么叫“与人消灾”？结合第一点和第二点，与人消灾的定义即是“做好领导交代的工作”。那么，做好领导交代的工作有哪些前提？A，理解领导的意图。方式有：复述一下领导的要求，并确认时间、质量、甚至是工作开展的节奏。B，过程中保持反馈。不管是进展顺利与否，都应该即是沟通进度，并确认方向无误。有困难更应该第一时间知会，同时有解决方案或备选方案。C，完成后汇报。很多人经常忽略这一点，所以经常会有那个case我前两天就发给您了啊这样的反馈。请注意一点：工作的完成不是以完成本身为衡量标准，而是以完成工作汇报并得到评价为衡量标准。4，上级永远没有错，是你错了。这句话太危言耸听了吗，我们严谨地说一遍：在业务方面，不涉及人格侮辱（如上级骂你是傻逼）、不涉及职业道德（如领导让你做假账）、不涉及越权，领导永远不会错。你的报表做完了，你发给他了，然后他没看。错过了提交的时间。不是他错了，是你错了，因为你没催他。你催他了，他还是没看。不是他错了，是你错了，因为你没催他第二遍。你催他第二遍了，他还是没看。不是他错了，是你错了，因为你没有在他不忙的时候，站在他电脑旁边告诉他，今天不看就来不及了，您现在抽空看看吧，然后哪有问题我再去改。这才叫职业化的态度。所以，职场根本不存在你说的明明是别人的错误却怪在我头上这种情况，而是你还没有足够成熟到持有一个最职业化的态度。5，你永远没有错，是上级错了。只有一种情况是这样，那就是当你们共同面对更大的领导的时候。好吧，最终，这个报表他还是没有看。这时更上一级的领导怪罪下来，在面对更上级领导的时候，你没错，是你领导的错。因为所有的下级责任在更上级领导的面前，都是你直接上级的责任。但是回到他办公室，只有你两个人的时候，还是你的错。6，不要要求上级完美。作为一个新人，上级不是你控制圈和影响圈的事物。你改变不了上级，只能改变自己。即使你换一个上级，你还是要学着改变自己才能成长。同时上级有七情六欲，有八小时之外的工作影响心情，有个人好恶有情绪起伏。请你好好体会上级的要求，而不是自顾自的觉得自己很正确。所以你现在要做的，不是据理力争，也不是逆来顺受。而是，把上级当做你在追的一个女孩子一样，去好好体会他对你什么感受，什么评价，尽量的去改正，尽量的去完善自己，尽量的去和他保持步调一致，这才是你最重要的事情。－－－－－以后是高阶的回答，职场菜鸟可忽略－－－－－－7，管理你的上级。之前提到，上级对你的评级可能是主观的。同时，上级通常扮演两个角色，一个是裁判员，一个是教练员。你要让他成为你的教练员，而不是裁判员，因为前者是自己人，后者是外人。怎么办？a，天下武功，唯快不破。领导交代的任务要迅速完成，迅速汇报，迅速检验质量。交给你一个报告，告诉你思路了，你两天出结果，他就是教练员，因为那有他的思路；你一周出结果了，他就是裁判员，因为他早忘了这件事。b，拿出80%的精力去应对20%最重要的事情。我没办法教你什么重要，给个小提示吧：如果某项工作上级问了你第二次，那多半是重要的事情，请最快速度完成。c，和他一起面对他的上级，帮助他完成上级领导给的任务。这个说得太高阶了，不具体谈方法了，还需要自己体会。8，多余的话。九零后开始洪水一样的涌入职场，我们也开始关注九零后的情绪，并尽量在这个过程中不要贴标签。但是孩子们，总是“既要薪情，又要心情”。从小又成长在相对自由、开放的的环境下，处处要求公平和尊敬。在归因方法上，经常归咎为明明是别人工作上的问题，而很少反思和思考自己。总觉得：我有能力有想法但是就是发挥不出来，甚至觉得“怀才不遇”。退一万步想，若是每个问题都是别人的问题，可能你的业绩评判方面会有好的影响。但是你会有进步吗？或者说，未来你的进步全部来自于成功经验的积累？没有日省吾身的精神，你自己能进步吗？所以还请各位职场新人，首先，要秉持一个“最职业的心态”，然后我们再来谈，什么叫“有思考的执行”，什么叫“实现自己的想法”。

2014年个人分析微报告

发表于 2015-01-01 | 更新于 2019-01-20

马上就要过完这2014年了，往年都有一个长长的个人总结，今年就来个微报告吧，算是对这一年的一个回顾。

现在我躺在南岳祝融峰上封寺的帐篷的睡袋里，回顾这一年让我很温暖，旁边还有一条大大的木鱼陪着我，等我明年醒来迎接新一年的曙光。

我相信明年会是更好的一年，去年的总结我说要抓住该抓住的，是的，2014年我无论是爱情上还是事业上，能抓住的我都没有放过，可回想起来觉得自己还是有些糊涂，糊涂的一年，没有把爱情修成正果，还失去了两个要好的朋友，没有把事业带上一个新的台阶，反而还被领导约谈多次。

2014年的收入较2013增加了50%，但是依然是负债累累，依然是没房没车没存款的屌丝男，没怎么出去旅行，也没给自己买什么大件，可账本上记下的一比比开销告诉我，钱就是这么被花光的。

今天也是我阳历的三十岁虚岁生日，想想从爸爸身体里出来都三十年了，对父母依然是没啥回报，十分惭愧，他们也许盼着我早日成家，早日立业，可他们依然只是在盼着。

我不算是一个虔诚的佛教徒，但是，我明天会早早起来，去给圣帝点上一柱香，祈祷父母、妹妹今后能够平平安安，身体健康，开心过好每一天；祈祷自己能够在2015年心想事成；祈祷我的朋友们步步高升，财源广进。

匆匆流年

发表于 2014-12-16 | 更新于 2019-01-20

趁着今天不上班和好太阳，去大学校园逛了一圈，猛然发现自己已经离开校园6年了，而这6年似乎没办好任何事情，工作一成不变，女朋友却一直在换。

连自己该如何反思都不知道了，对未来一片迷漫，想要的东西多，比如想换工作自己创业，想找女朋友然后结婚，想去东南亚来一次半个月的旅行，想买一辆自己的车。

也许我一直对自己的各种定位不对，一屌丝，安安稳稳娶个老婆结婚不就是了嘛，工作反正的公务猿，混混日子也就一生过去了。

可我的天性是自由的，不喜欢被束缚，不喜欢被绑架。

2015年终于到了三十而立的年纪，我是需要改变还是继续沉淀下自己，在这个十字路口有些迷漫。

Android使用GAE代理和PC上goagent翻墙

发表于 2014-07-07 | 更新于 2019-01-20

今天就介绍如何使用安卓上的GAE代理结合goagent来翻墙教程分为两个A,B两个阶段：A: goagent GAE部署(此部分晚上教程很多，请自行google)B.安卓手机上配置GAE代理1.首先当然是下载GAE代理了，您无法在play store下载，可以在任意的第三方市场下载，如果实在找不到，也可以留言给我，我再上传到论坛。2.设置：安装好后，打开GAE代理进行设置，见下图：注意在代理地址那里填写你自己的appid，比如之前的inexus(如果能用的话)那么请在代理地址填写：https://yourappid.appspot.com/fecth.py(注意yourappid要被你自己的id替换掉，比如inexus)端口的话我测试可以随便填写：我这里写的是8087.如果您想开机就启用GAE代理，就在自动链接打勾。想使用GAE代理访问GOOGLE PLAY的话，电子市场代理打勾,另外我选择的是分应用代理，也就是个别应用才使用代理，你可以点击去设置下，比如勾选chrome,google play使用代理的。没有选择全局代理，因为翻墙的速度并不具备优势，不需要所有的访问都用代理。另外，https代理不要打钩，我测试会让国外的网站都无法访问（除了twitter.com）。然后最后记得 打开代理开关，不然它是不会运行的。这样设置好并使得GAE运行后，你打开chrome就可以看到开篇的那个图了，使用google play就可以下载更多的应用了。
