2014年个人分析微报告

发表于 2015-01-01 | 更新于 2019-01-20

马上就要过完这2014年了，往年都有一个长长的个人总结，今年就来个微报告吧，算是对这一年的一个回顾。

现在我躺在南岳祝融峰上封寺的帐篷的睡袋里，回顾这一年让我很温暖，旁边还有一条大大的木鱼陪着我，等我明年醒来迎接新一年的曙光。

我相信明年会是更好的一年，去年的总结我说要抓住该抓住的，是的，2014年我无论是爱情上还是事业上，能抓住的我都没有放过，可回想起来觉得自己还是有些糊涂，糊涂的一年，没有把爱情修成正果，还失去了两个要好的朋友，没有把事业带上一个新的台阶，反而还被领导约谈多次。

2014年的收入较2013增加了50%，但是依然是负债累累，依然是没房没车没存款的屌丝男，没怎么出去旅行，也没给自己买什么大件，可账本上记下的一比比开销告诉我，钱就是这么被花光的。

今天也是我阳历的三十岁虚岁生日，想想从爸爸身体里出来都三十年了，对父母依然是没啥回报，十分惭愧，他们也许盼着我早日成家，早日立业，可他们依然只是在盼着。

我不算是一个虔诚的佛教徒，但是，我明天会早早起来，去给圣帝点上一柱香，祈祷父母、妹妹今后能够平平安安，身体健康，开心过好每一天；祈祷自己能够在2015年心想事成；祈祷我的朋友们步步高升，财源广进。

2013年个人分析总结报告

发表于 2013-12-27 | 更新于 2019-01-20 
2012年的“末日总结”，我在恐慌中写完，今年，我已涅槃重生，不在意世界万象的事物，万物由我心流出，真正做回了我自己。去年我说： 在我眼里，每一天都是末日，我认真地把每一天过好，过得有声有色、多姿多彩，过得充实、有感悟，过得不会让未来的自己悔恨现在的自己。我想我已经做到了！       2013年的总结，我一直没想好怎么写，因为这一年过得太累，身心疲惫，头脑中的倦意一直提醒我应该好好睡一觉，但是每天似乎都是零点过后才睡，很多人问我为什么叫“小午”，其实是午夜的意思，作为摩羯座，我喜欢黑夜，喜欢在夜深人静的时候思考。所以，今年的年度关键词：累！第一篇   身心疲惫         疲惫的一年就要过去了，2013年，我没做什么事，也做了很多事，没做事情是因为经常没去单位上班，一周五天平均上班2-3天，大家不要鄙视我，我也是有苦衷的，与其浪费时间去乡镇混日子，我还不如把时间腾出来做自己的事情，这个工作我在去年的总结中就说了：工作给我带来的恐慌就是我不能明白工作还能给我带来什么！ 既然这个工作是没有目标和目的的，那我何必呢！做了很多事情是因为一直在做着些似乎没有什么意义的兼职，做过医院的SEO，做过旅行社网站，做过领队，做过微博营销，做过淘宝客，做过广告设计...还做着户外网站的管理，很多事情，都是在我没去上班的时候做得，虽然挣钱不多，但是这些都是我喜欢做的事情，做起来无比激情！虽然忙碌，但是心里踏实！虽然很累，但是很快乐！        感情的事情也让我疲惫。谈了一段恋爱，浪漫地开始，理性地结束，爱过、彷徨过、后悔过，虽然以分手告终，但是我认真去谈了，结局并不总要。重要的是在这段恋爱过程中我学到了什么，也许我的感情观还不够成熟，所以造成了这样的结局，感情是一个双方都需要慢慢探索的过程，最终能不能步入婚姻的殿堂并且有幸福的婚姻生活，不是靠缘分，不是靠激情，而是靠双方的互相体谅、互相进步、相互扶持。感情的事情让我疲惫，是因为年纪的已经摆在这里了，29了，后年就30，三十而立，对于一个男人来说，家庭责任感比事业重要，一个幸福的家庭能让事业更加顺利，可反观下现在的自己，事业家庭都还是没有一点点到来的意思。继续努力吧，虽然疲惫，但是也要打起万分精神去寻觅、去追寻！         各种累让我身心疲惫，所以我也是需要释放的，旅游和娱乐活动成了发泄途径，也是我一年来最大的花费。大年初一就出发了，注定了今年这一年都是在外奔波的，初一至初七在凤凰、铜仁，初九去了海南，去了两次香港，爬了多次南岳岣嵝峰，还有无数的小地方。旅行，是一场心灵的修行，用心灵的窗户去感受世间美景，用双脚去丈量这个世界，用心去沐浴情感冷暖！人应该去旅行，在年轻的时候，趁着有脾气装潇洒，有本钱耍个性，离开睁眼闭眼看见的城市，逃离身边的纷纷扰扰，找一个让心里安静和干净的地方，让自己变得跟水晶一般透明，然后拍一些美得想哭的照片，留给老年的自己。一个地方呆久了，就会觉得腻，所以，旅行就是到自己不腻的地方去感受生活，不需要豪华舒适的酒店，也不需要美味可口的饭菜，要做的只是带着爱，用爱去探索未知的事物、未知的情感、未知的自己。每一次旅行都是一场六感的饕餮大餐，是一次心灵的洗礼。带着爱去旅行，你会发现不一样的自己！旅行拍了不少照片，年底做了明信片寄给了很多人，相信大家都会喜欢的。第二篇    数据统计     每年的数据分析都会让大家为之一惊，我喜欢用数字来衡量事务，数据化的东西在我的脑海里会无比清晰。先是社会化媒体数据:1.微博粉丝属性分析2.微博粉丝性别分布3.粉丝地域分布4.我的最佳粉丝个人财务分析(我相信也没多少人看得懂吧)，今年养成了记账的习惯，知道自己的每一块钱花在了什么地方，也随时提醒自己要开源节流。下面是一些财务统计数据：这是一年来的开支情况，其中最大的一块是休闲娱乐，包含旅游，淘宝，打牌等；其次是食品酒水，包含吃饭，零食，烟酒等；再就是行车交通费。这三大块占了总花费的70%资金流动表是显示现金、银行账户、信用卡、支付宝间的资金流动情况。其中信用卡还有欠款，说明我是负资产。收支趋势图，总体来看年初旅游花费较大，后面基本每个月收支平衡。项目汇总表，这是单独列出来的一些项目，如旅游花费7280元（不包含购物），麻将字牌扑克等共输掉1038，看来手气不是不错，而是相当的烂！还有一些数据，如手机话费，互联网搜索，地理位置统计等，我就不一一公布了，留做子孙后代们去研究吧。第三篇    新的起点        2014我希望能有一个新的起点，累点没关系，没有钱也没关系，只要自己能抓住一个目标，能够为了这个目标去努力去奋斗。没有好工作，没关系；没有爱人，也没关系！我要努力做好我自己。抓住该抓住的一切！

这就是爱情，吗？

发表于 2013-11-06 | 更新于 2019-01-20

在这个年代谈论爱情似乎有点太昂贵了，可爱情还是真真切切存在的。

据网络统计，两个男女只要坠入爱河，平均只需224条微博、163条短信、70条Facebook动态、37封电子邮件和30通电话就够了。而分手，36%使用电话通知、27%使用短信，13%使用网络。

分手对我来说，早已没有那痛彻心扉的悲伤，平静接受、过度、开始下一段恋情….

我一直认为自己早已不会去爱，不懂得如何去爱，或者说不会付出全部的爱，把爱留给自己，这是我作为一个单身唯一能做的，爱自己，爱生活。

选择自己爱的人，那叫爱情，选择爱自己的人，那叫生活。也许我太爱生活，所以不知道如何去选择自己爱的人，也就木讷地不懂得什么叫爱情，在我的思想里，更多的是生活，这么说，我是应该结婚了，因为结婚的人才会有这样的思想。

让我们谈一场不赶时间的恋爱

发表于 2013-10-31 | 更新于 2019-01-20
很多年前有一个女同事，年纪不小，其貌不扬，却嫁的风光无限。不少人背后窃窃私语讨论，讨论她到底用了什么手段。我也一直纳闷，那个时候，我还处在以为女人只有整容瘦身才有资格钓金龟婿的精神境界里。
不久我就发现了她的好处。哪里有好吃又不贵的地方。她知道！办什么事要去哪里，办什么证应该怎么弄。她知道！家里布置的温馨又不失高雅，全是她一手包办。在那些整日叽叽喳喳的小女生面前，她简直就是天降神兵，移动的百科全书，有了她遇到各种鸡毛蒜皮完全可以淡定的自动元神归位。
她在自己的生活中找到了最合适的位置，只要这地球还在继续转动，她就有被需要的价值和意义，就像必不可少的114，就像GOOGLE地图，就像百度。所以，她遇到了那个需要她的男人，他们一拍即合结为连理。

这样的故事，或许被很多人不屑。她们会问，那么爱情呢，那么激情呢。可是不久后，你就会发现，当你哭哭啼啼的疑问爱情到底是什么，激情的保质期到底有多久的时候，她还在那里，淡然踏实的生活着，并且比大多数人感觉到幸福。
不管你愿意或者不愿意，每个人都需要找到一个定位。这个位置能给予你太多安定，远比财富或者男人能让你安心。
她知道自己不够美、不够媚、不够优秀，但是她够耐心够用心够心定神怡，所以她不纠结、不混乱，坚定朝着那条路走下去，到最后竟然令其他人大跌眼镜。
这就像一只稳定的绩优股，不暴涨也不暴跌，每天涨一点点，最后黑马般杀出。
可是很多人等不到这样的春天，她们急着恋爱，急着抓住什么，急着结婚。为什么急呢？她们也不知道，因为大家都很急，所以她们不能不急，惶惶不可终日，到了最后，忘记了自己是谁。

这个国度，太多物质的生长周期都被人为缩短，生猪几个月就可出栏，一只鸡活不过三个月。热恋期缩短，婚姻的平均寿命变短。只有婴儿必须呆在母体里十个月，千百年不曾改变。可是，再这样下去，是不是婴儿也可以有五个月速成的呢。
让我们试着慢慢来。
时光能赋予你的绝然超乎你的想象。
从一个不知所谓的女孩，变成某个人人生里的天降神兵。
把一个青涩害羞的男孩，变成可以真正令你仰赖的大神。
其实我们缺乏的不是够好够对够靠谱的那个人，而是最后的最后，大家变成了彼此都需要的那一个人。如果你给他多一点时间，他也多给你一点时间，多好。
