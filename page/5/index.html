Android手机安装Google服务框架方法

发表于 2012-12-17 | 更新于 2019-01-20
很多国内行货的android手机、或是刷的自制精简ROM，都没有google的服务框架，根本不能登录google的各项服务，android手机如果没有google的服务的话那和功能机没有很大区别了。这就需要我们自己给手机安装google的服务框架，享受google的各项服务带来的便利。

google基础服务包包涵以下几个文件：
GoogleServicesFramework.apk——这个是google基础服务框架，这个是最基础的，google的服务都基于它，一般情况下，要用任何google服务必需先安装它。
ContactsSyncAdapter.apk——这个是google联系人同步服务接口，安装了这个，账户与同步里面的联系人同步才可用，不然点同步就只是一闪而过，根本不执行。
CalendarSyncAdapter.apk——这个是google日历同步服务接口，安装了这个，账户与同步里面的日历同步才可用。
GoogleLoginService.apk——这个就是google登录认证服务。用于各种应用的登录认证。
GoogleQuickSearchBox.apk——这个就是google的快速搜索。
ChromeBookmarksSyncAdapter.apk——这个就是google chrome的书签同步服务。
Gmail2.3.6.apk——这个就是gmail。
Google play.apk——这个就是google商店。

我们要先root，然后将GoogleServicesFramework.apk和GoogleLoginService.apk两个文件拷贝到/system/app/目录下，然后将文件权限改为rw-r–r–,然后再根据自己需要选择安装其他的如Gmail,google play等基础服务。

感恩节！

发表于 2012-11-22 | 更新于 2019-01-20

感恩节（英语：Thanksgiving Day）是美国和加拿大共有的节日，原意是为了感谢上天赐予的好收成。在美国，自1941年起，感恩节是在每年11月的第四个Thu ，並从这一天起将休假两天。像中国的春节一样，在这一天，成千上万的人们不管多忙，都要和自己的家人团聚。加拿大的感恩節則起始於1879年，是在每年10月第二个Mon，与美国的哥伦布日相同。

今天是感恩节，我们要懂得感恩。“感恩，并不是要去感谢某个给你帮助的人，而是一种心态，一种感受生活中充满着爱的心态。”这是我去年的感恩节发的微博，相信有爱，相信生活中充满爱，你就会过得更开心。我把这段话写在了给EX的信中，我相信，爱是维系一段感情，一段婚姻，一个家庭的主要精神力量。对长辈的感恩，就是孝顺，对爱人的感恩，就是爱情，对朋友的感恩，就是友情。

感恩！是一种爱的体现，怀着一颗大爱的心，你才知道感恩，才知道这个世界有多少事情值得你去珍惜~~

反日游行—官方默许的民怨泄愤口

发表于 2012-09-20 | 更新于 2019-01-20

过去一周，自日本将钓鱼岛国有化以来，中国各地出现了各类大小规模的游行，从呼吁抗日，到抵制日货，再到打砸抢，游行，似乎已经失去了他原本的意义。这一游行浪潮在9.18日达到顶峰，全国大中小城市都举行了反日游行示威，与以往各类游行示威不同，这次官方的态度似乎不是很明确，各地没有出现大批量武警镇压，也没有催泪瓦斯、水枪等防暴工具，难道官方就这么放心广大民众？而官方这样的态度还是在各地都有打砸抢现象出现的情况下发生的，是什么让官方这一次如此缄默，从中央到地方，没有一个领导站出来表态的。这一次，民众终于成了主角，官方的成了配角。

从官方的默许态度来看，我有如下分析：

1.政府出面支持的话，不利于缓和中日高层关系；

2.政府出面禁止的话，会造成官民矛盾升级，而此时是中共十八大即将召开的关键时期，激化矛盾不利于社会和谐稳定；

3.各路媒体采取了不报道、不参与、勤删帖的态度，说明了中央不想把事情扩大化，让事态快速燃烧，快速熄灭。就像是给民众一个“树洞”，让你们说，让你们发泄，然后就没然后了；

4.各地警察、武警、特警，居然在游行中保持了应有的“风度”，没有和民众发生冲突，只是在默默地维护秩序，给游行示威“擦屁股”，可见，政法系统肯定是有高层授意的，要不如此大的城市混乱，市委书记都是要收到批评的。

第一点：要有爱。我所说的爱不仅仅是指爱情，而是指内心要有爱，这种爱包含爱生活，爱自己，爱家人，爱所有可以爱的人，是一种能够包容一切的大爱，也就是说一个女孩子要懂得尊重、包容、平等，而且做人要真诚，我想，能够这样做到的，至少是个善良的人了，与人为善，这是我自己的为人观。对人尊重，你才会和我相敬如宾，你才不会让我在朋友面前丢面子；你会包容，才能在与我吵架后重归和好，才能接纳我所有的缺点，才能理解我的好与坏；懂得平等，你才会设身处地为人着想，你才不会矫情。做人真诚，你才能让我看到真正的你。

第二点：要会生活。话说结婚就是两个人凑在一起过日子，既然是过日子，那就要会生活，我不需要你会精打细算，但是你至少不能大手大脚花钱；我不需要你会做山珍海味，但是你至少要会几个拿手好菜；我不需要你会相夫教子，但是你至少会懂得教孩子做一个善良、正直的好人；我不需要你气质高雅、品味高端，但是你至少要有几个健康的兴趣爱好。其实生活本就应该平平淡淡，我不希望 你给我整出什么惊喜，一点小事就一哭二闹三回娘家的，这样的生活会很恼心。

第三点：要会欣赏自己。要懂得欣赏自己，认为自己是最美丽的，话说自信的女人最美丽，其实我这人对外貌要求不是那么多，只要这个女孩和我走出去人家会说：这一对好搭哦！所有我会有身高163CM以上的基本要求。其实我觉得，女人七分靠打扮，一个认为自己很美的女人，一定会精心地打扮自己，能把地摊货穿出走红地毯的范儿那就是最自信的了。

第四点：要会欣赏我。虽然说我不是什么风度翩翩，也不是什么才高八斗，如果你爱我，就要学会欣赏现在和过去的我，我就是我，独一无二的我，我有很多缺点，但是你更多地是要学会欣赏我的优点，我相信我优点发出的光芒会掩盖我的缺点的。

白富美会生活，会欣赏自己，但是肯定看不上我这屌丝男，而且从来都是被动接受爱的，哪里会知道什么是爱。PASS….

最新官方网站 https://magiclantern.fm 可以在这里下载最新的固件

今天给500D升级固件的，从1.0.9升级到了1.1.0，在搜索的时候无意发现了佳能的第三方固件Magic Lantern(魔灯)，支持
佳能EOS系列的5D Mark II, 550D, 60D, 600D, 50D, 7D, 700D, 500D ,5D Mark II,5D Mark III 等型号，该第三方固件实现了许多高级功能，

如：

音频：可禁用AGC和数字滤波器，音频电平显示，手动音频控制，选择使用输入源（内部，内部+外部，外部立体声，平衡），通过USB端口监听。
曝光助手：斑马纹，伪色，直方图，波形，中心点测光。对焦工具：对焦峰值，录制时焦点画面放大，陷阱对焦，变焦，跟焦，焦点叠加合成，对焦图，面部优先检测模式下画面放大。
视频助手：比特率控制（QScale或CBR），视频日志记录（EXIF元数据），缓冲区溢出或4GB限制后自动重新开始录制，剩余时间显示，LiveView显示时隐去所有叠信息，改变模式拨盘上的视频模式键的位置。

包围：包围曝光，焦点叠加合成。
延时拍摄：间隔时间拍摄（照片和视频），无需快门驱动的静音照片; 包围曝光及合成。
天文夜间摄影：很长时间（高达8H）的Ｂ门曝光。
信息显示：焦点和DOF信息，CMOS温度，快门计数器和时钟。
对于闪客：从-10到+3 EV的闪光曝光补偿（取决于相机）。
电源管理：LiveView 模式下的关闭显示，快速调整LCD背光级别。
有趣的东西：狭缝扫描照片。

第一步 准备工作

去官网 https://builds.magiclantern.fm 下载最新的魔灯固件。
*佳能卡片机也有一个类似的第三方固件叫CHDK：http://chdk.wikia.com/wiki/CHDK
将电池充满电移除任何相机附件（比如电池手柄或外接闪光灯）。请务必使用SD/SDHC卡进行首次安装，SDXC卡不能用于首次安装。
请务必使用一款高质量的读卡器；用USB线连接相机是无法进行安装的。
如果你的相机固件版本和Magic Lantern要求的不一致，请先移步官网进行固件升级。
去官网下载适应的固件版本，官网的固件包内有详细的升级教程，按照步骤来就对了。

佳能550D / T2i / Kiss X4：所需固件版本1.0.9
佳能60D：所需固件版本1.1.0佳能50D： 所需固件版本 1.0.8 (Windows, Mac)
佳能600D / T3i / Kiss X5：所需固件版本1.0.1
佳能500D：所需固件版本1.1.1
佳能 5D Mark II ：所需固件版本 2.1.1 (Windows, Mac)

将相机模式调至Manual (M)位置（50D, 550D, 600D）或Bulb (B)位置 （60D）。
将相机恢复到出厂设置。

第二步 安装从相机内对存储卡进行格式化 （低级格式化）。

将下载的zip文件解压至SD卡根目录。启动固件升级。
开机并选择固件版本 1.x.x。当你看到绿色的确认画面时，重启相机，大功告成。

###(中文的)功能介绍

  • 2.1 音频：模拟增益可调，数字增益可调。可禁用AGC和数字滤波器，音频电平显示，手动音频控制，选择使用输入源（内部+外部，内部，外部立体声，平衡），通过USB端口监控音频。
  • 2.2 LiveView：斑马，伪色，直方图，波形，任意位置点测光，构图辅助框图，边缘检测，局部放大。
  • 2.2.1 斑马Zebra：在过曝和欠曝区域显示斑马纹。
  • 2.2.2 伪色False Color：可以选择使用某种着色图来评价曝光。
  • 2.2.3 直方图Histogram：不解释了。。。
  • 2.2.4 波形Waveform：一个很神奇的用来显示亮度信息分布的图。
  • 2.2.5 构图标记CropMarks：各种裁剪图印在屏幕上，方便构图。
  • 2.2.6 点测光Spotmeter：可在屏幕上任意移动的小点，并显示亮度。
  • 2.2.7 峰值对焦Focus Peak：手动对焦利器。数图边缘检测，用一阶或二阶导数判断边缘，用条纹显示。
  • 2.2.8局部放大Magic Zoom；另一个手动对焦利器。想放大的部位可移动，放大框上下绿线为合焦提示，未合焦的局部图像可裂像显示。
  • 2.3 视屏高级工具；比特率可调，帧率可调，HDR2.3.1可调比特率BitRate：佳能原厂的压缩比出片的话差不多330MB/min，可根据不同需求调整为原bitrate的0.1-3.0倍。并且可变比特率VBR的程度也可调。2.3.2自动重启Movie Restart：佳能原厂拍片满4GB后会自动停止，开启后可自动重启拍摄。
  • 2.3.3数码变焦DigitalZoom：600D可用1x..10x
  • 2.3.4视频HDR：圣诞节出的版本上有这个功能，所以我还没用过。看样片player.vimeo.com/video/33987353感觉细节表现力不错。原理是把间隔的两帧视频分别处理成欠曝和过曝（或者正常和欠曝），后期时再用软件处理融合。Youtube上有各种用AE的处理教程。
  • 2.3.5帧率可调：同样出现在圣诞节出的版本上，据说1080P可设置4-32fps,720p可以设置4-60fps，满足各种后期需要。
  • 2.4 摄影Shoot：HDR，Timelapse，B门，声控快门，运动检测，反光镜预升。
  • 2.4.1HDR Bracketing: 佳能原厂自带拍3张的不同曝光。ML此功能至多拍9张，间隔EV可调，一次快门自动低速连拍完成所有frame。不过仍需后期合成，在unix系统下貌似可以直接运行脚本观看。
  • 2.4.2间隔曝光Intervalometer：用于拍摄微速摄影Timelapse，可控制间隔几秒钟拍一张照片，附带bulb ramping功能以弥补所摄环境光线的缓慢变化（比如从白天到黑夜）。我瞬间觉得先前为了拍《秒速交大》而买的定时快门线真是白买了。关于timelapse可详见我上一篇日志。
  • 2.4.3超级B门Bulb Timer：原厂快门貌似至多拍30秒吧。。ML可以选择1秒到8小时，霸气侧漏有木有，虽然几小时CMOS都要烧掉了。
  • 2.4.4声控快门Audio RemoteShot：通过声音的分贝数来控制触发快门，阈值可调。当周围不太吵闹时，对着相机大吼一声即拍，可用于自拍。
  • 2.4.5运动检测Motion Detect：根据明度变化判断是否有运动，以此触发快门，阈值可调。
  • 2.4.6反光镜预升Mirror Lockup
  • 2.5 曝光选项Expo: ISO，白平衡，白平衡偏移，快门，光圈，照片风格，闪光补偿，Exp.Override2.5.1ISO调节：可更精细地调节ISO，比如可用125。
  • 2.5.2白平衡调节：可直接调节色温
  • 2.5.3 Exp.Override：主要用于视频拍摄的曝光优化，比如可用1/8000s、ISO12800，24fps时可用1/25s。
  • 2.6对焦Focus：2.6.1陷阱对焦Trap Focus：可用于陷阱法大鸟。半按快门情况下，若合焦即触发快门。
  • 2.6.2对焦点选择Focus Patterns：可选择各种组合对焦点pattern。
  • 2.6.3手动跟焦Follow Focus：可通过按键来调整对焦距离，可用于拍摄视频时的跟焦。
  • 2.6.4对焦渐变Rack Focus：设置了对焦的终点、步长之后，可从起点自动一步步变焦到终点。可用于视频拍摄中的前景到背景切换，步长小的时候过渡很自然。笔者在试用这个功能的时候出现过死机和err70的情况，请拔插电池再重启。
  • 2.6.5对焦渐变堆Stack Focus：与上条差不多，是变一个对焦距离拍一张照片，会得到一堆照片。
